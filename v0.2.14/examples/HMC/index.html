Hamiltonian Monte Carlo Sampler

Previously we computed the log likelihood of the untuned kernel parameters of the GP, $-1.285$. We now also perform approximate inference over said kernel parameters using the No-U-Turn Sampler - Hamiltonian Monte Carlo(NUTS-HMC) Sampler (paper) provided by AdvancedHMC.jl. We start of by loading necessary packages.

using AdvancedHMC, Distributions, ForwardDiff

We define a function which returns log-probability of the data under the GP / log-likelihood of the parameters of the GP.

function logp(params; x=x_train, y=y_train)
    kernel = ScaledKernel(
        transform(
            Matern52Kernel(),
            ScaleTransform(exp(params[1]))
        ),
        exp(params[2])
    )
    f = GP(kernel)
    fx = f(x, 0.01)
    return logpdf(fx, y)
end
logp (generic function with 1 method)

Set the number of samples to draw and warmup iterations

n_samples, n_adapts = 2_000, 1_000
initial_params = rand(2)
2-element Array{Float64,1}:
 0.5908446386657102
 0.7667970365022592

Define a Hamiltonian system

metric = DiagEuclideanMetric(2)
hamiltonian = Hamiltonian(metric, logp, ForwardDiff)
Hamiltonian(metric=DiagEuclideanMetric([1.0, 1.0]))

Define a leapfrog solver, with initial step size chosen heuristically

initial_ϵ = find_good_stepsize(hamiltonian, initial_params)
integrator = Leapfrog(initial_ϵ)
Leapfrog(ϵ=0.4)

Define an HMC sampler, with the following components

  • multinomial sampling scheme,
  • generalised No-U-Turn criteria, and
  • windowed adaption for step-size and diagonal mass matrix
proposal = NUTS{MultinomialTS, GeneralisedNoUTurn}(integrator)
adaptor = StanHMCAdaptor(MassMatrixAdaptor(metric), StepSizeAdaptor(0.8, integrator))
StanHMCAdaptor(
    pc=WelfordVar,
    ssa=NesterovDualAveraging(γ=0.05, t_0=10.0, κ=0.75, δ=0.8, state.ϵ=0.4),
    init_buffer=75, term_buffer=50, window_size=25,
    state=window(0, 0), window_splits()
)

Run the sampler to draw samples from the specified Gaussian, where

  • samples will store the samples
  • stats will store diagnostic statistics for each sample
samples, stats = sample(
    hamiltonian,
    proposal,
    initial_params,
    n_samples,
    adaptor,
    n_adapts;
    progress=false
)
samples_mat = reduce(hcat, samples)';

Average log-marginal-probability of data with posterior kernel parameter samples sampled using ESS. We can observe that there is significant improvement over exact posterior with default kernel parameters.

plt = scatter(x_train, y_train, label="Train Data")
scatter!(plt, x_test, y_test, label="Test Data")
for params in @view(samples[(end-100):end,:])

            Matern52Kernel(),

    f = GP(opt_kernel)
