Variational Inference

Sanity check for the Evidence Lower BOund (ELBO) implemented according to M. K. Titsias's Variational learning of inducing variables in sparse Gaussian processes

elbo(fx, y_train, f(rand(7)))
-563.4926598610715

We will be using Optim.jl package's LBFGS algorithm to maximize the given ELBO.

using Optim

Create a helper function for optimization. It takes in the parameters (both variational and model parameters) and returns the negative of the ELBO.

function optim_function(params; x=x_train, y=y_train)
    kernel = ScaledKernel(
        transform(
            Matern52Kernel(),
            ScaleTransform(exp.(params[1]))
            ),
        exp.(params[2])
    )
    f = GP(kernel)
    fx = f(x, 0.1)
    return -elbo(fx, y, f(params[3:end]))
end
